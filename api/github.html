Greetings

Github is a popular web-based hosting service for version control. The website was founded by Tom Preston-Werner, Chris Wanstrath, P.J. Hyett, and Scott Chacon in 2008, and is now owned by Microsoft. Github is a place where users can upload any files for hosting and sharing, but it is mainly used for software developers to upload their codes and collaborate with one another through open source contributions. Many great software were created due to communities of people making constant changes and optimizing the software. Anyone can create an github account and make their own repositories to show off their work to the world!

Create a Github Account

The first thing that we needed to do before getting the API key is to create a Github account here at this link. Then, we must register an OAuth application by filling in the lines given on Github's website.

Once we are done registering for an application, we can view our OAuth Client ID Key, and Client Secret Key, both of which will be required to use all of Github's API. Let's move on to the next page where we can start setting up our website to use the Github's API with node.js and express-handlebars.
